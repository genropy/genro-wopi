# genro-wopi

WOPI (Web Application Open Platform Interface) implementation for the Genropy framework.

## Status

**Development Status: Pre-Alpha**

This project is in early planning phase. No implementation code yet.

## Description

genro-wopi provides WOPI protocol support for integrating document editing capabilities
with Collabora Online, LibreOffice Online, and other WOPI-compatible editors.

## Features (Planned)

- WOPI discovery endpoint
- File operations (GetFile, PutFile, Lock, Unlock)
- Document editing integration
- Access token management
- Collaborative editing support

## Installation

```bash
pip install genro-wopi
```
